# Student Grade Management system
съдържание:Сорс код на C#, включващ класовете Students, Subjekt, Gradecalculator и главната програма Program.
 # Кратко описание
конзонлно приложение на C# което реализира система за управление на оченическиоценки,използвайки обектно-ориентиран подход
# какво прави проектът?
-Създава и съхранява профили на ученици с техните имена.
-Управлява оценки чрез динамична колекция List. 
- Изчислява среден успех за всеки ученик автоматично.
- извежда информация в конзолата за по-лесно проследяване.
# Използжани технологии 
-Език нз C#.
# Как е стартиран проектът 
-Проектът се отваря в Visual Studio.
-Стартира се като Console Application.
-програмта се пуска с клавишната комбинация Ctrl+F5 или чрез бутона Start.
# Структура 
-Students-Информация за ученик и неговите уценки. 
-Subject-Учебен предмет,свързан с конкретен ученик.
-GradeCalculator-логика за изчисляжане на успеха и валидация.
-Program-Стартож файл,който демонстратира работата на системета.
